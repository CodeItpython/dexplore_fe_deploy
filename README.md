# Dexplore Front-end Deploy

## 💡개요 💡
기존에 만들었던 것의 최적화 및 UI/UX 수정으로 최종 결과물 제출을 위한 Deploy 리포지토리를 개설.

## 💡실행 방법 💡
### 🚀 배포 🚀
``` shell
$ npm build 
```
해서 빌드된 파일을 배포하면 됨.

### 🚀 실행 🚀
```shell
$ npm run
```
해서 실행시킨 후 테스트 진행

## 💡디렉토리 설명 💡
- prev_backup
  - 기존에 작성한 소스코드 참고용으로 붙여넣어둠.
- public
  - 배포하는 파일이 담긴 디렉토리
- src
  - 작업하는 공간

## 💡작업 방식 💡
- 개인 브랜치 개설 후 브랜치 위에서 작업할 것.
- 메인에는 절대 커밋 금지
